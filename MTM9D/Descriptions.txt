	ОПИСАНИЕ ВЕРСИЙ MTM9D

1. MTM9D - графическое приложение-заготовка из "QT Designer" без функционала.

2. MTM9D_2 - Приложение позволяет работать с устройством через COM порт, выводит
	последнее измерение в окно приложения, записывает данные в таблицу excel, 
	приложение не устойчиво, связь rs485 не устойчива, запись в таблицу не устойчива.

3. MTM9D_3 - Изменена архитектура приложения(рефакторинг), что позволило
	обрабатывать событие закрытия окна, приложение не устойчиво, связь rs485
	не устойчива, запись в таблицу не устойчива.

4. MTM9D_4 - Рефакторинг, приложение не устойчиво, связь rs485
	не устойчива, запись в таблицу не устойчива.

5. MTM9D_5 - Добавлен график, приложение не устойчиво, связь rs485
	не устойчива, запись в таблицу не устойчива.

6. MTM9D_6 - завершенное приложение для 1-го датчика.

7. MTM9D_7 - переработка приложения для нескольких датчиков
